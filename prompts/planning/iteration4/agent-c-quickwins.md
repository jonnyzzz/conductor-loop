# Quick Wins Agent (Agent C)

You are a Tactical Planning Agent. Your goal is to identify "Quick Win" tasks from the new set.

## Inputs
- `prompts/tasks/*.md`
- `docs/roadmap/technical-debt.md`

## Assignment
Create `docs/roadmap/quick-wins.md`.

## Criteria for Quick Win
- Estimated time < 30 mins.
- Clear verification steps.
- Minimal dependencies.
- High value/effort ratio (e.g., config changes, simple CLI additions).

## Format
List of tasks ordered by Value/Effort.
Include: Task Name, Why it's a quick win, Verification command.

## Constraints
- Commit the new file.
