# Task: Hello World

Please create a simple greeting message that demonstrates Conductor Loop is working correctly.

## Requirements

1. Output a markdown file with a friendly greeting
2. Include the current timestamp
3. Mention the agent type being used
4. Confirm the task completed successfully

## Output Format

Create a markdown document with:
- A heading "Hello from Conductor Loop!"
- A brief description
- The current timestamp
- The agent type
- A success message

## Example Output

```markdown
# Hello from Conductor Loop!

This is a simple demonstration of task execution.

The current timestamp is: 2026-02-05 11:51:54 UTC
The agent type is: codex
The task completed successfully.
```

## Instructions

Write your output to `output.md` in the current working directory.
